# Enable GitHub Pages - 手动步骤

## 🚀 立即启用步骤

### 1. 访问仓库设置
**URL**: https://github.com/shaguoerai/seo-readability-tool/settings/pages

### 2. 配置Pages
1. **Source**: 选择 "Deploy from a branch"
2. **Branch**: 选择 "main" 分支
3. **Folder**: 选择 "/ (root)" 文件夹
4. **点击 "Save"**

### 3. 等待部署
- 部署通常需要 **1-2分钟**
- 刷新页面查看状态
- 状态变为 "Published" 表示成功

### 4. 访问工具
**URL**: https://shaguoerai.github.io/seo-readability-tool/
